# pawroman-dotfiles

## Requirements

### ZSH

- nerd fonts

- exa
- kubetail
- rg
- thefuck
- stow

Post-install configuration

- set `GITSTATUS_DIR` in `~/.zshrc.local-pre`
- Put local customizations to `~/.zshrc.local-pre` and `~/.zshrc.local-post`

## Install

Use the `stow-all.sh` script to create symlinks.
